# Overview

This app is built with React, bundled with Webpack, and uses Gulp to compile Sass. It displays Instagram, Twitter, and Vine posts from an API that contain the *#*iflyalaska hashtag. 

# Technologies Used
React  
React-Router  
Webpack  
SCSS  
Gulp
